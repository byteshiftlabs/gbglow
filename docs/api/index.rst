Types & Constants
-----------------

Integer Types
~~~~~~~~~~~~~

+-----------+-------------------+---------------------+--------------------------------------+
| Type      | Definition        | Range               | Usage                                |
+===========+===================+=====================+======================================+
| ``u8``    | ``std::uint8_t``  | 0 to 255            | Register values, memory bytes        |
+-----------+-------------------+---------------------+--------------------------------------+
| ``u16``   | ``std::uint16_t`` | 0 to 65,535         | Addresses, register pairs            |
+-----------+-------------------+---------------------+--------------------------------------+
| ``u32``   | ``std::uint32_t`` | 0 to 4,294,967,295  | Cycle counters                       |
+-----------+-------------------+---------------------+--------------------------------------+
| ``s8``    | ``std::int8_t``   | -128 to 127         | Signed offsets (JR, ADD SP)          |
+-----------+-------------------+---------------------+--------------------------------------+
| ``Cycles``| ``std::uint32_t`` | 0 to 4,294,967,295  | M-cycle counting                     |
+-----------+-------------------+---------------------+--------------------------------------+

Memory Constants
~~~~~~~~~~~~~~~~

+-------------------+-------------+------------------------------------------------+
| Constant          | Value       | Description                                    |
+===================+=============+================================================+
| ``MEMORY_SIZE``   | 0x10000     | Total addressable memory (64KB)                |
+-------------------+-------------+------------------------------------------------+
| ``VRAM_SIZE``     | 0x2000      | Video RAM (8KB)                                |
+-------------------+-------------+------------------------------------------------+
| ``WRAM_SIZE``     | 0x2000      | Work RAM (8KB)                                 |
+-------------------+-------------+------------------------------------------------+
| ``OAM_SIZE``      | 0xA0        | Object Attribute Memory (160 bytes)            |
+-------------------+-------------+------------------------------------------------+
| ``HRAM_SIZE``     | 0x7F        | High RAM (127 bytes)                           |
+-------------------+-------------+------------------------------------------------+
